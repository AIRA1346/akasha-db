# 📚 AKASHA Global Works Database

이 저장소는 로컬-퍼스트 아카이빙 애플리케이션 **아카샤(AKASHA)**의 공식 작품 사전 데이터베이스입니다. 
누구나 자유롭게 데이터를 추가 및 수정하여 전 세계 모든 서브컬처 및 대중 문화의 작품 메타데이터를 풍성하게 채워나갈 수 있습니다.

---

## 🛠️ 작품 등록 데이터 규격 (JSON)

작품 정보는 `works_registry.json` 파일에 아래와 같은 형식으로 저장됩니다.

```json
  "작품_고유_식별키": {
    "workId": "작품_고유_식별키 (소문자, 언더바 권장)",
    "title": "공식 작품 제목 (한국어 권장)",
    "category": "manga | animation | game | book | movie | drama (6종 택 1)",
    "domain": "subculture | generalCulture (서브컬처 여부에 따라 택 1)",
    "creator": "원작자 / 제작사 / 감독 등",
    "releaseYear": 출시 또는 연재 시작 연도 (숫자만),
    "description": "작품에 대한 대략적인 줄거리 및 소개 요약",
    "tags": ["장르1", "태그2", "키워드3"],
    "posterPath": "공식 이미지 원격 주소 (http/https로 시작하는 이미지 직링크) 또는 null"
  }
```

### 📌 기여 규칙 및 카테고리/도메인 구분법

#### 1. category (매체 소분류)
* `manga`: 만화책, 웹툰 등
* `animation`: 애니메이션, 극장판 애니 등
* `game`: 모바일, 콘솔, PC 게임 전체
* `book`: 일반 도서, 소설, 라이트노벨 등
* `movie`: 영화 (애니메이션 영화 제외)
* `drama`: 드라마, 시리즈 등

#### 2. domain (대분류)
* `subculture`: 매니아 지향 만화/애니메이션 계열, 일본 라노벨 테마 미소녀/수집형 게임(예: 블루 아카이브, 니케) 및 관련 영화/서브컬처 소설 등.
* `generalCulture`: 일반 소설(예: 히가시노 게이고), 학습/대중만화(예: 수학도둑), 대중적인 AAA 게임 및 e스포츠 게임(예: 마인크래프트, 리그 오브 레전드, 엘든 링), 일반 대중 영화 및 드라마 등.

## 🤝 기여하는 방법 (How to Contribute)

1. 이 저장소를 **Fork**합니다.
2. `works_registry.json` 파일을 열어 새로운 작품 정보를 맨 아래에 추가합니다.
3. 데이터의 문법적 오류가 없는지 JSON 검증기 등을 통해 체크한 후, **Pull Request (PR)**를 보냅니다.
4. 검수 완료 후 메인 브랜치에 병합되면, 전 세계 아카샤 사용자들의 앱 대시보드 사전에 실시간 동기화로 자동 반영됩니다.
